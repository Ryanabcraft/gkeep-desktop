# Google Keep Desktop

App desktop leve que abre o [Google Keep](https://keep.google.com) em uma janela limpa, sem abas de navegador.

- ✅ **Login automático** — usa seus cookies do navegador, sem precisar logar de novo
- ✅ **Multiplataforma** — Windows, Linux e macOS
- ✅ **20+ navegadores** — Chrome, Edge, Brave, Firefox, Opera, Vivaldi e mais
- ✅ **Código aberto** — Python puro, ~10 MB
- ✅ **Sem telemetria** — não coleta nenhum dado

## Download

| Plataforma | Download |
|------------|----------|
| **Windows** | [`GoogleKeep-Setup.exe`](https://github.com/Ryanabcraft/gkeep-desktop/releases/latest) — Instalador Inno Setup |
| **Linux / macOS** | [`googlekeep_app.py`](googlekeep_app.py) — Rode com `python3 googlekeep_app.py` |

## Como usar

### Windows
1. Baixe o instalador `.exe` da [última release](https://github.com/Ryanabcraft/gkeep-desktop/releases/latest)
2. Execute e siga os passos
3. Pronto! O Google Keep abre em janela limpa

### Linux / macOS
```bash
wget https://raw.githubusercontent.com/Ryanabcraft/gkeep-desktop/master/googlekeep_app.py
python3 googlekeep_app.py
```

Requer Python 3.6+ e um navegador instalado.
